## 🚀 **Quick Start**

```bash
git clone https://github.com/jamoran1356/dashboard-pro-next
cd dashboard-pro-next
pnpm i  # Fast deps
pnpm dev  # localhost:3000
```

## 📋 **Features**
- **UI Moderna**: Tailwind gradient, cards shadows, responsive grid.
- **Sidebar Nav**: Dashboard/Usuarios/Ajustes buttons.
- **Stats Mock**: KPIs "5 Usuarios", "$2,450 Ingresos", "↑12% Crecimiento".
- **Graph Placeholder**: Actividad semanal.
- **pnpm Ready**: Faster installs.
- **Zero Deps Complex**: Static, no DB/auth.
